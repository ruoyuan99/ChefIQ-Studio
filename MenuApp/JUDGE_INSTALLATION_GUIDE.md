# 评委安装指南 - Chef iQ Android 应用

本指南帮助评委在 Android 手机上安装 Chef iQ 应用。

---

## 📱 安装步骤

### 步骤 1: 允许安装未知来源应用

在安装 APK 之前，需要允许手机安装来自未知来源的应用：

1. 打开手机 **设置**
2. 找到 **安全** 或 **隐私** 选项
3. 启用 **"允许安装未知来源的应用"** 或 **"安装未知应用"**
   - 不同手机品牌位置可能不同：
     - **小米/红米**: 设置 → 应用设置 → 授权管理 → 应用安装管理
     - **华为/荣耀**: 设置 → 安全 → 更多安全设置 → 外部来源应用下载
     - **OPPO/一加**: 设置 → 其他设置 → 设备与隐私 → 安装外部来源应用
     - **vivo**: 设置 → 安全 → 未知来源
     - **三星**: 设置 → 锁定屏幕和安全 → 安装未知应用
     - **其他品牌**: 通常在 "安全" 或 "隐私" 设置中

### 步骤 2: 下载 APK 文件

**方法 A: 通过链接下载**
1. 在手机浏览器中打开提供的下载链接
2. 点击下载按钮
3. 等待下载完成

**方法 B: 通过二维码下载**
1. 使用手机相机或扫码应用扫描二维码
2. 在浏览器中打开链接
3. 点击下载按钮
4. 等待下载完成

### 步骤 3: 安装应用

1. 下载完成后，点击通知栏的下载通知
   - 或打开 **文件管理器** → **下载** 文件夹
2. 找到 `chef-iq-*.apk` 文件
3. 点击 APK 文件
4. 点击 **"安装"** 按钮
5. 等待安装完成

### 步骤 4: 打开应用

1. 安装完成后，点击 **"打开"** 按钮
   - 或在应用列表中找到 **"Chef iQ"** 图标
2. 点击打开应用
3. 首次打开可能需要几秒钟加载

## ⚠️ 常见问题

### Q: 提示 "无法安装" 或 "安装被阻止"

**解决方案：**
1. 确保已启用 "允许安装未知来源应用"（见步骤 1）
2. 某些手机需要为特定应用（如浏览器）单独授权：
   - 在设置中找到您使用的浏览器（Chrome、Firefox 等）
   - 允许该应用安装其他应用

### Q: 下载后找不到 APK 文件

**解决方案：**
1. 打开 **文件管理器** 应用
2. 进入 **下载** 或 **Download** 文件夹
3. 查找文件名包含 "chef-iq" 或 "apk" 的文件

### Q: 安装时提示 "解析包时出错"

**可能原因：**
1. APK 文件下载不完整
2. 手机存储空间不足
3. Android 版本不兼容（需要 Android 5.0 或更高版本）

**解决方案：**
1. 重新下载 APK 文件
2. 清理手机存储空间
3. 检查手机 Android 版本（设置 → 关于手机 → Android 版本）

### Q: 应用打开后闪退

**解决方案：**
1. 重启手机
2. 清除应用缓存（设置 → 应用 → Chef iQ → 存储 → 清除缓存）
3. 重新安装应用

---

## 📋 系统要求

- **Android 版本**: Android 5.0 (API 21) 或更高版本
- **存储空间**: 至少 100MB 可用空间
- **网络**: 首次使用需要网络连接（用于加载数据）

---

## 🔄 更新应用

如果收到新版本的 APK：

1. 下载新的 APK 文件
2. 直接安装（会覆盖旧版本）
3. 无需卸载旧版本
